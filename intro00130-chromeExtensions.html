3  3. Chrome Extensions

A chrome extension is a piece of software that extends the capabilities of the Chrome browser. ## How to install a Chrome Extension

First make sure that you have the Chrome browser installed.

Then, click here for info about how to install and use a chrome extension.

https://www.youtube.com/watch?v=g92v8bP8dKo

  • Try installing an using the “google dictionary” chrome extention

  • Try installing an using the “google translate” chrome extention

3.2 Using other browsers

Chrome extensions may work in other browsers as well, but Chrome is the best option to make sure that the extension works as intended.

3.3 Developing your own Chrome Extensions

Using modern AI, it is often possible to develop your own custom Chrome extensions. We will not go into that here, but the adventurous are free to experiment on your own.

While under development a Chrome extension consists of several files in a folder. Once the coding of the Chrome extension has been completed, the entire folder with all of the necessary files can be compressed into a single zip file. It is this zip file that can be upladed to the Chrome Extension store.

3.3.1 “unpacked” Chrome extensions

While under development, a Chrome extension can still be installed in Chrome without needing to ZIP up the folder. Follow the instructions here to use a chrome extension that is still under development. You will need to

  1. unzip the folder if it is zipped

  2. navigate to the url chrome://extensions

  3. Click the “developer mode” switch

  4. Click “load unpacked”

Chrome extensions are created using standard web development languages (i.e. HTML, CSS, Javascript).

You can follow the instructions in this video:

https://www.youtube.com/watch?v=yNFwFQrc27Q
